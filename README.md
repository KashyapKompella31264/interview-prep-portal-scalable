# Interview Preparation Portal – Backend

A full-stack coding interview platform backend built using Java and Spring Boot.

## Features
- JWT-based authentication with role-based access (ADMIN, INSTRUCTOR, STUDENT)
- REST APIs for managing users, courses, questions, and submissions
- Secure backend architecture with modular service design
- Integrated code execution support using Docker containers
  

## Tech Stack

- Java, Spring Boot
- PostgreSQL / MongoDB
- JWT Authentication
- Docker
- RESTful APIs

## Project Structure
- controller/ – API endpoints
- service/ – business logic
- repository/ – database access layer
- config/ – security & JWT configuration

## How to Run
1. Clone the repository  
2. Configure application.properties  
3. Run the Spring Boot application
